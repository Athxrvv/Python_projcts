## Projects

### 1. API Fuzzer - Security & Stability Testing Tool
**Location:** `api_fuzzer/`

A comprehensive automated API testing tool that sends high volumes of randomized and malformed inputs to REST endpoints to identify security vulnerabilities and stability issues.

**Features:**
- Automated fuzzing with randomized payloads
- Security testing (SQL injection, XSS, command injection)
- Edge case simulation
- Dynamic payload generation with Faker
- Structured logging and response validation
- Comprehensive JSON reports

**Tech Stack:** Python, Requests, Faker, Pytest
